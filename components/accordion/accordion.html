Stiiks Challenges is a design system and component library built around CSS custom properties and SVG iconography. It follows a minimal, asset-driven approach with centralized theming.

How do I use the components?

Each component is self-contained with its own HTML and CSS files. Simply reference the base.css for global styles and the component-specific CSS for styling. All colors are defined as CSS custom properties in config.css.

What about accessibility?

This accordion uses native HTML <details> and <summary> elements, which provide built-in keyboard navigation (Space/Enter to toggle) and screen reader support. The browser automatically manages ARIA states and focus handling.

Can I customize the colors?

Yes! All colors are defined as CSS custom properties in config.css. You can easily modify the color palette by changing these variables. The component uses semantic color names like --neon-yellow, --green, and --purple from the design system.
